Class Description Usable tags
clearfix Use for clearing of float left/right elements inside of element with this class any block tag
margin-0 It will set margin of element to 0px. any tag
padding-0 It will set padding of element to 0px. any tag
hidden It will set display: none style. It is used in sliding boxes for example. any tag
visible It will set display: block style. It can be used for displaying of hidden elements. any tag
float-left It will set float: left to an element. any tag
float-right It will set float: right to an element. any tag
textleft it will set text-align: right. any tag
textcenter It will set text-align: center. any tag
textright It will set text-align: right. any tag
alignleft it will set float: right and margin: 0 0 20px 20px. img
alignright It will set float: right and margin: 0 20px 20px 0. img
