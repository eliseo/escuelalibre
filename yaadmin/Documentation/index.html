This template can be fixed or fluid of your choice, with two columns or one. Sidebar has a fixed width, and the content area is the remaining free area. All of the information within the main content area is nested within a div with an id of "page". The sidebar's (column #2) content is within a div with an class of "col2". The general template structure is the same throughout the template. Here is the general structure.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

#page a {
	color: #someColor;
}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

#wrapper #page a {
	color: #someColor;
}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

This theme imports two PHP files and CSS & JS Compressor module.

  1. GAPI - Google Analytics PHP Interface (gapi.class.php)
  2. Visitors last 30 days (ajax.php)

  1. For statistics of visitors, I used a template GAPI - Google Analytics PHP Interface.

  2. For the use you need to make to the file /php/ajax.php data for authentication and Google report ID for the selected account.

    <?

$email = "yourmail@gmail.com";
$password = "PASSWORD";
$report_id = 34583510; // $report_id is the Google report ID for the selected account

.......

    That's all you need to use this functional my theme.

    This file is not recommended output data means echo and print, the script should return data in JSON format.

  3. Usual connection script files looks like this: 
    <script type="text/javascript" src="js/jquery-1.4.4.min.js"></script>
<script type="text/javascript" src="js/jquery-ui-1.8.7.custom.min.js"></script>
<script type="text/javascript" src="js/yaadmin.js"></script>
<script type="text/javascript" src="js/custom.js"></script>
    The connection is through a script looks like this: 
    <script type="text/javascript" src="js/js.php?js,jquery-1.4.4.min,jquery-ui-1.8.7.custom.min,yaadmin,custom"></script>
